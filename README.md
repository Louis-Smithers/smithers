# Smithers

Marketing site for Smithers — Canadian operations and automation consultancy.

## Stack

- TanStack Start (React 19 + Vite)
- Tailwind CSS v4 + shadcn/ui
- TypeScript
- Bun

## Local development

```bash
bun install
bun run dev
```

The app runs at http://localhost:5173 (or whatever Vite picks).

## Build

```bash
bun run build
```

## Deploy

Hosted on Railway, deploys automatically from GitHub on every push to `main`. See [`DEPLOY.md`](./DEPLOY.md) for the one-time setup.

## Notes

- The landing page is a shared `<SmithersLanding locale="en"|"fr" />` component in `src/routes/index.tsx`.
- French route: `src/routes/fr.tsx` renders the same component with `locale="fr"` and FR-localised meta + FAQ JSON-LD.
- Locale auto-detection: on first `/` visit with no saved preference, if `navigator.language` starts with `fr` the user is redirected to `/fr`. The `EN / FR` pill in the header persists choice to `localStorage["smithers-lang"]`.
- All copy lives in `src/i18n/en.ts` and `src/i18n/fr.ts` (Quebec French). Types are in `src/i18n/types.ts`. `src/i18n/useT.ts` returns the active translation object based on locale.
- **Entry point**: `src/main.tsx` bootstraps the TanStack Router client (hydrates if SSR shell is present, creates root otherwise). The old SPA entry (`src/App.tsx`) has been removed.
- **TanStack Start wiring**: the site is a proper TanStack Start app — not a plain Vite SPA. `src/server.ts` is the server entry, `src/router.tsx` builds the router, and `src/routeTree.gen.ts` is auto-generated by the `@tanstack/router-plugin` Vite plugin on each build. Do not hand-edit `routeTree.gen.ts`.
